# Lira — чек-лист после получения архива

Когда оформишь ИП и подключишь платёжку, по этому чек-листу запустишься в
коммерческую продажу. Каждый шаг помечен временем и стоимостью.

---

## 1. Юр.лицо (ИП) ~ 1 неделя, бесплатно

1. **Госуслуги → «Открыть ИП»** (если ещё нет КЭП — нужна электронная подпись,
   её делают бесплатно в банке вместе с расчётным счётом).
2. **ОКВЭД:**
   - **62.01** — разработка ПО (основной)
   - **62.02** — консультирование в IT
   - **47.91** — розничная торговля по интернету (для физ.боксов)
   - **47.99** — прочая розничная торговля вне магазинов
3. **Налоговый режим:** **УСН «Доходы» 6%** (или 1% если в регионе льгота
   для IT) — самый простой.
4. После регистрации откроется расчётный счёт (Тинькофф / Точка / Сбер —
   бесплатно для ИП).

## 2. Поддержка-почта ~ 5 минут, 0–500 ₽/год

- **Yandex 360 free** (1 ящик): https://360.yandex.ru → подключи свой домен.
- Или **Mail.ru для бизнеса** (бесплатно): https://biz.mail.ru
- Адрес: `support@<твой-домен>.ru`

## 3. Запись на 152-ФЗ в реестр операторов ПД ~ 30 минут, бесплатно

Сайт **РКН → «Электронные сервисы» → «Уведомление об обработке ПД»**:
https://pd.rkn.gov.ru/operators-registry/notification/form/

Поля:
- Цель: оказание услуг по подписке, информирование о продуктах
- Категории субъектов: клиенты (физ.лица 16+)
- Категории ПД: ФИО, телефон, email, telegram_id, данные о цикле
- Способы обработки: автоматизированная
- Меры защиты: шифрование канала (TLS), ограничение доступа

После подачи — в течение 30 дней попадёшь в реестр операторов ПД.
Это **обязательно** для продаж в РФ.

## 4. Реквизиты в коде ~ 5 минут

Отредактируй **ОДИН файл**: `web/legal/footer.js`

```javascript
window.LIRA_SELLER = {
  name:    'ИП Иванова Ирина Сергеевна',
  inn:     '7700000000',
  ogrn:    '320770000000000',
  email:   'support@твой-домен.ru',
  address: 'г. Москва',          // юр.адрес можно скрыть до уровня города
  tg:      '@lira_support',      // основной канал поддержки
};
```

Эти значения подтягиваются автоматически:
- внизу страницы (в футере),
- в Политике конфиденциальности,
- в Договоре-оферте,
- в Контактах,
- в Telegram-боте (через ENV `SUPPORT_EMAIL`, `SUPPORT_TG`).

## 5. Платёжный шлюз ~ 1–3 дня

**Рекомендую ЮКассу:**
1. Зарегистрируйся как ИП на https://yookassa.ru
2. Загрузи документы (паспорт, ИНН, ОГРНИП) — модерация 1–3 дня.
3. Получишь `shopId` и `secretKey`.
4. Открой `backend/api/main.py`, найди функцию `create_payment_link()` —
   заменим заглушку на реальный вызов ЮКассы (попроси меня — это 30 мин
   работы, нужен только твой `shopId`).
5. **54-ФЗ:** ЮКасса сама отправит чек в ОФД и продублирует клиенту на
   email — отдельная онлайн-касса не нужна.

Альтернативы: Тинькофф-Касса, Робокасса, CloudPayments. Все интегрируются
~одинаково.

## 6. Yandex.Metrika ~ 5 минут, бесплатно

1. https://metrika.yandex.ru → «Добавить счётчик» → введи свой домен.
2. Скопируй ID счётчика (число вида `123456789`).
3. Открой `web/index.html`, найди:
   ```javascript
   var ID = '00000000';  // TODO: paste your Yandex.Metrika counter ID
   ```
   замени `00000000` на свой ID.
4. Деплой:
   ```bash
   ssh root@IP "cd /opt/lira && docker compose restart api"
   ```

В настройках счётчика обязательно:
- **Webvisor — выключи** (это запись действий пользователя, требует
  отдельного согласия по 152-ФЗ).
- **Карта кликов** — выключи по той же причине.
- **Электронная коммерция** — включи, чтобы видеть конверсию.

## 7. Sentry (мониторинг ошибок) ~ 5 минут, бесплатно

1. https://sentry.io → Sign up → создай проект «Browser JavaScript».
2. Скопируй DSN (вид: `https://abc@o123.ingest.sentry.io/456`).
3. Открой `web/index.html`, найди:
   ```javascript
   var DSN = '';  // TODO: paste Sentry DSN here
   ```
   вставь свой DSN в кавычки.
4. Деплой как в п. 6.

Лимит free-плана: 5000 ошибок/мес. Тебе хватит на первые 2000–3000 юзеров.

## 8. Мониторинг uptime ~ 3 минуты, бесплатно

1. https://uptimerobot.com → Add New Monitor:
   - **Type:** HTTP(s)
   - **URL:** `https://<твой-домен>/health`
   - **Interval:** 5 minutes
2. В разделе «Alert Contacts» добавь Telegram-уведомления через
   `@uptimerobotbot`.
3. Если сайт упадёт — придёт сообщение в Telegram в течение 5 минут.

## 9. Бэкапы — настроены автоматически

`setup.sh` уже добавил cron-задачу:
```
17 3 * * * /usr/local/bin/lira-backup.sh
```

Каждую ночь в 03:17 сохраняет SQLite-снапшот в `/opt/lira/backups/`,
хранит 14 дней. Скрипт можно запустить вручную:
```bash
ssh root@IP "/usr/local/bin/lira-backup.sh && ls -la /opt/lira/backups/"
```

**Рекомендация:** через месяц-два настрой off-site копию на
Yandex.Object Storage (бесплатно до 1 GB):
1. https://cloud.yandex.ru → Object Storage → создай bucket `lira-backups`.
2. Установи `rclone` на VPS, настрой профиль `yandex`.
3. В `/usr/local/bin/lira-backup.sh` добавь после `gzip`:
   ```bash
   rclone copy "$BACKUP_DIR/app.db.$DATE.gz" yandex:lira-backups/
   ```

## 10. Иконки и брендинг

В `web/icons/` лежат placeholder-иконки. Когда будет финальный логотип,
сгенерируй набор иконок здесь: https://realfavicongenerator.net и положи
их в `web/icons/`, перезапиши:
- `apple-touch-icon.png` (180×180)
- `icon-192.png`
- `icon-512.png`
- `icon-maskable-512.png` (с safe-зоной 410×410 в центре)

## 11. PWA на iOS — установка пользователями

Когда сайт работает по HTTPS:
1. Открыть `https://твой-домен.ru` в **Safari** (не Chrome!).
2. Кнопка «Поделиться» → «На экран Домой» → «Добавить».
3. Иконка появится на главном экране, открывается без адресной строки.

Для **Android** аналогично через Chrome → меню «...» → «Добавить на главный экран».

## 12. APK для Android (опционально)

Если захочешь распространять через Google Play или установочный файл —
TWA-обёртка (Trusted Web Activity) уже есть в архиве (см. APK от прошлой
итерации). Перед публикацией:
- Подпиши `assetlinks.json` (см. https://developers.google.com/digital-asset-links).
- Положи его в `web/.well-known/assetlinks.json` (Caddy раздаст автоматом).
- Тогда TWA откроется без браузерной полоски сверху.
